## 📋 Tổng quan

Dự án frontend được xây dựng với kiến trúc hiện đại, sử dụng Vite để build nhanh, Tailwind CSS cho styling và Alpine.js cho interactivity.

### 🎯 Tính năng nổi bật

- ⚡ **Vite** - Build cực nhanh với HMR
- 🎨 **Tailwind CSS** - Utility-first CSS framework
- 🔥 **Alpine.js** - JavaScript framework nhẹ
- 📱 **Fully Responsive** - Tương thích mọi thiết bị
- 🌙 **Dark Mode** - Hỗ trợ giao diện tối
- 🔒 **Authentication** - Xác thực và phân quyền
- 🚀 **PWA Ready** - Progressive Web App support
- 📊 **Charts & Analytics** - Biểu đồ thống kê
- 🔌 **WebSocket** - Real-time updates
- 🧪 **Testing** - Unit tests với Vitest

## 🛠 Cài đặt

### Yêu cầu

- Node.js >= 18.0.0
- NPM >= 9.0.0

### Các bước

```bash
# Clone repository
git clone <repository-url>
cd 02_web_frontend

# Cài đặt dependencies
npm install

# Copy environment file
cp .env.example .env

# Chạy development server
npm run dev

# Build production
npm run build
