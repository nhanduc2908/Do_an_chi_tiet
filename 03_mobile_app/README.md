# Security Evaluation System - Mobile App

## 📱 Giới thiệu

Ứng dụng mobile cho Hệ thống Đánh giá An toàn Thông tin, hỗ trợ cả Android và iOS.

## 🚀 Tính năng

- Đăng nhập / Đăng ký
- Xác thực vân tay (Biometric)
- Dashboard theo vai trò (10 roles)
- Quản lý thiết bị
- Nhân bản màn hình
- WebSocket realtime
- Báo cáo thống kê

## 🛠 Cài đặt

### Yêu cầu
- Flutter 3.16+
- Dart 3.2+
- Android Studio / VS Code

### Các bước

```bash
# Clone dự án
cd C:\xampp\htdocs\security_evaluation_system\03_mobile_app

# Cài dependencies
flutter pub get

# Chạy ứng dụng
flutter run

# Build APK
flutter build apk --release
